What is Statik

Statik is a system currently in development for collecting information on servers running your Bukkit plugins.

We a group of plugin developers who have gathered together to create this system. We are creating this because the current system that is used by the majority of plugins collecting information is unreliable, the website breaks regularly and the administrator has been AWOL for quite a while now.

What data is collected

The following data is anonymously collected about servers:

  • Java Version
  • Operating System
  • System Architecture
  • Number of System Cores
  • Amount of System Memory
  • Plugin Name
  • Statik Version
  • Server Mod (Bukkit, Spigot etc...)
  • Server Location
  • Authentication Mode
  • Server Player Count
  • Plugin Version
